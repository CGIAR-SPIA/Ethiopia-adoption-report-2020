# Ethiopia-adoption-report-2020
Welcome to this repository, which contains the replication files for **[Shining a Brighter Light: Evidence on Adoption and Diffusion of CGIAR-Related Innovations in Ethiopia](https://iaes.cgiar.org/spia/publications/shining-brighter-light-comprehensive-evidence-adoption-and-diffusion-cgiar)**.  

This repository provides the code and supporting data needed to reproduce the tables and figures in the Ethiopia synthesis report.  

The repo includes:
- `data/`
- `scripts/`
- `outputs/`
